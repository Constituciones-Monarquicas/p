/inicio/

En 1978 cuando se creo nuestro sistema tributario, no existía el desarrollo normativo que había ni los recursos que se daba para pillar a la gente, en aquel momento se creo un impuesto sobre la renta.

Lo que se hizo fue una actividad censal y de control, para saber que tiene cada persona, y así cobrárselo, lo que hace es que este impuesto nació también por una proporcionalidad, porque si una persona tiene mas patrimonio que otro, este impuesto es muy polémico y nació para controlar mejor que se le cobraba así nació el IP  como solución a los errores al IRPF.

Este impuesto se creo con carácter temporal, pero se quedó para siempre, hemos de tener claro que en este país se da que los impuestos temporales se hacen permanentes, este impuesto temporal se prorrogo cada año hasta el año 91 cuando se aprueba la ley actual, y se hace permanente.

En 2008 se decidió que nadie en España pagara este impuesto, pero en vez de derogarlo lo que hicieron fue a nivel estado bonificarlo al 100% por lo que quedo como un muerto en vida. Llegamos a finales de 2010 y para recaudar por la crisis se tma como una de las opciones suprimir de forma provisional la bonificación, esta situación se fue prorrogando hasta 2019, que se establecio que se suprimia su carácter temporal, y se le dejó plenamente vigente, a nivel del estado no hay bonificación del 100%.

Pero a las CCAA se les permiten competencias, y algunas de estas hoy lo tienen bonificado al 100% por lo menos es la situación de hoy.
